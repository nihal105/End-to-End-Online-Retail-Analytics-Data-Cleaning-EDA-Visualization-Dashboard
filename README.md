# End-to-End-Online-Retail-Analytics-Data-Cleaning-EDA-and-Visualization


##  Project Overview

This project presents a complete data analytics workflow on an online retail dataset, covering **data cleaning**, **exploratory data analysis (EDA)**, and **interactive dashboard creation**.
The goal is to transform raw transactional data into meaningful business insights that support decision-making.

##  Objectives

* Clean and preprocess raw retail data for analysis
* Perform exploratory data analysis to identify trends and patterns
* Analyze customer behavior, product performance, and revenue distribution
* Build an interactive dashboard for business insights

---

##  Dataset

* **Source:** Online Retail Dataset
* **Records:** 500K+ transactions
* **Time Period:** Dec 2010 – Dec 2011
* **Features Include:**

  * Invoice No
  * Stock Code
  * Description
  * Quantity
  * Invoice Date
  * Unit Price
  * Customer ID
  * Country

---

##  Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn)
* **Jupyter Notebook** (Data Cleaning & EDA)
* **Power BI** (Dashboard & Visualization)

---

##  Data Cleaning Steps

* Removed missing and null values
* Filtered out negative quantities (returns/cancellations)
* Removed duplicate records
* Converted data types (date, numeric fields)
* Created new features (Revenue, Month, Day, etc.)

---

##  Exploratory Data Analysis (EDA)

* Monthly revenue trends and seasonality
* Top-performing products by revenue
* Country-wise sales distribution
* Customer purchase behavior
* Revenue patterns by day of week

---

##  Dashboard Highlights

*  Total Revenue: £10.67M
*  Total Orders: 19,960
*  Unique Customers: 4,338
*  Average Order Value: £534

### Key Insights:

* The **UK dominates sales (~84%)**, indicating a strong domestic market
* **November shows peak revenue**, highlighting seasonal demand
* A small number of products generate a large portion of revenue
* International markets show **growth potential**, especially in Europe
